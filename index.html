One-time setup
You need a free Firebase Realtime Database URL to enable real-time sync.
1
Go to console.firebase.google.com and sign in.
2
Open your project, then go to Build → Realtime Database → Create database → Start in test mode → Enable.
3
Copy the database URL shown at the top, e.g.
https://your-project-default-rtdb.firebaseio.com
Please enter a valid Firebase Realtime Database URL.
