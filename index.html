🔥 One-time setup

BBQ Planner uses Firebase Realtime Database so friends share a live plan. Free, no credit card needed.

  1. Go to console.firebase.google.com → Add project
  2. Name it anything, skip Analytics, click Create
  3. Left sidebar: Build → Realtime Database
  4. Create Database → any region → start in test mode → Enable
  5. Copy your database URL e.g. https://my-bbq-default-rtdb.firebaseio.com
  6. Paste below and save

URL is stored only in your browser's localStorage.
